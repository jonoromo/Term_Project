Detailed Description

This file is uploaded to a Nucleo STM32 to control the "Brendan and the Jo(h)nathans" NERF turret.

The file is written to cooperatively multitask, using a thermal camera, DC motor with closed loop PI control, and servomotor.

main.task1_fun ( )

Accepts camera input via I2C.

Camera processing. Takes a picture after 5 seconds, when the target is frozen. Filters data, omitting rows and columns on the edges of the frame to focus on the target.

◆ task2_fun()

main.task2_fun ( )

Motor control: both panning axis and trigger-pull servo.

After initializing motor, the panning axis spins 180 deg. Both flywheels are turned on. Panning motor turns gun according to setpoint from thermal camera. Servo pulls trigger and returns to neutral position.
