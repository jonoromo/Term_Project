Detailed Description

This class implements a motor driver for an ME405 kit.

The user can specify a PWM value and change the speed of the motor. Changing the sign of PWM changes the direction of the motor.

Creates a motor driver by initializing GPIO pins and turning off the motor for safety.

Parameters
en_pin: Pin on motor controller corresponding to pin PA_10 on the Nucleo. Setting high enables the motor.
in1pin: Pin on motor controller corresponding to pin PB_4 on the Nucleo. Used to send PWM signal to motor controller in negative direction.
in2pin: Pin on motor controller corresponding to pin PB_5 on the Nucleo. Used to send PWM signal to motor controller in positive direction.
timer: Sets timer for PWM. Set to 20,000 Hz to avoid an annoying hum.
channel_pos: Channel number corresponding to the pin controlling the PWM output, channel 2.
channel_neg: Channel number corresponding to the pin controlling the PWM output, channel 1.

Positive values cause torque in one direction, negative values in the opposite direction.
